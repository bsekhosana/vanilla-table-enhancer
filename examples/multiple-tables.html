Multiple Tables Example

This example shows how you can enhance multiple tables on the same page. Each table has its own independent search, sort, and pagination controls.

Code Used

// Method 1: Enhance specific tables by ID
VTE.enhance('#usersTable');
VTE.enhance('#productsTable');

// Method 2: Enhance all tables at once
VTE.enhance('table');

// Method 3: Enhance multiple tables with an array
VTE.enhance(['#usersTable', '#productsTable']);
